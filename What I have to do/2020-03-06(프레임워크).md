# 프레임워크
#

> 프레임워크의 종류와 왜 사용하는지에 대해서 공부하고, 기본적인 프레임워크 사용법 익히기

* 프레임 워크를 이용해서 간단한 프로젝트 만들기(엄청 간단해도됨)

#

# 배운 내용/ 알게 된 내용

> 프레임워크는 이미 짜여진 틀안에서 그 기능에 특화되어 간단하게 쓸 수 있게 나온 형태이다.
> 프레임워크를 사용함으로써 편하면서 질좋은 프로그램을 개발할 수 있다.

# Framework 종류

* python
  * Django(장고- 웹 프레임워크)
   > 인스타그램에서 만든 웹 프레임 워크
  * Flask(플라스크- 웹 프레임워크)
   > 페이스북에서 만든 웹 프레임 워크
   
* javascript
  * node express(웹 프레임워크(백엔드))
  * 리액트 (웹 프레임워크(프론트엔드))
